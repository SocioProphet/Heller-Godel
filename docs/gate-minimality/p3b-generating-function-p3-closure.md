## 1. Statement of closure

P3.b closure verifies the four S-requirements from `HG-MTH-015` against the arity-three Fuss-Catalan generating function:

```text
C_3(x)=1+x C_3(x)^3
```

using the constructor-shape statistic on the grammar constructed in `HG-MTH-014`:

```text
C_A^(3) = (A -> A -> A -> A) -> A -> A
T ::= x | t(T,T,T)
```

Closure statement:

Under the constructor-shape statistic on alpha-classes of eta-long beta-normal de Bruijn lambda terms of type `C_A^(3)`, the proof-class / proof-family generating function is:

```text
C_3(x)=sum_{n >= 0} FC_n^(3) x^n
```

where:

```text
FC_n^(3)=1/(2n+1) * binom(3n,n).
```

This generating function satisfies:

```text
C_3(x)=1+x C_3(x)^3,
C_3(0)=1.
```

It is analytic at `x=0` with `C_3(0)=1` and has a finite positive radius of convergence. The existence of a positive radius is closed here by the analytic implicit-function argument in Section 5. The precise location of the dominant singularity belongs to P3.c / `HG-MTH-018` and is not promoted by this P3.b theorem.

## 1a. Formal theorem statement (theorem-grade promotion)

`HG-MTH-016` Theorem. Under the constructor-shape statistic on alpha-classes of eta-long beta-normal de Bruijn lambda terms generated by:


```text
C_3(x)=1+xC_3(x)^3.
```

Its coefficients are the arity-three Fuss-Catalan numbers:

```text
[x^n]C_3(x)=1/(2n+1) * binom(3n,n).
```

Proof: four components, each already within the normalized `HG-FND-002` generating-function discipline or guarded by this PR:

3. Local branch — `F(x,y)=y-1-xy^3` satisfies `F(0,1)=0` and `F_y(0,1)=1`, so the analytic implicit function theorem selects a unique analytic branch with `C_3(0)=1`.
4. Coefficient extraction — Lagrange inversion gives `[x^n]C_3(x)=1/(2n+1) * binom(3n,n)`, guarded by `tests/test_hg_mth_016_lagrange_inversion_guard.py`.

## 2. Verification strategy

This closure selects the combined V2 + V3 strategy from `HG-MTH-015`.

V2 establishes the functional equation structurally from the grammar:

```text
T ::= x | t(T,T,T).
```

The base normal form `x` contributes the constant term `1` under the constructor-shape statistic because it has zero `t`-nodes. Any non-base normal form is uniquely of the form:

```text
t(T1,T2,T3)
```

where `T1`, `T2`, and `T3` are normal forms generated by the same grammar. The top constructor contributes one `t`-node, and the three subterms contribute independently. Therefore the ordinary generating function obeys:

```text
C_3(x)=1+x C_3(x)^3.
```

V3 extracts the coefficients from this algebraic equation by Lagrange inversion. Let:

```text
Y(x)=C_3(x)-1.
```

Then:

```text
Y=x(1+Y)^3.
```

For `n >= 1`, Lagrange inversion gives:

```text
[x^n]Y = 1/n [u^(n-1)](1+u)^(3n)
       = 1/n * binom(3n,n-1).
```

The binomial form converts to the standard arity-three Fuss-Catalan form by:

```text
1/n * binom(3n,n-1)
= 1/n * (3n)! / ((n-1)! (2n+1)!)
= (3n)! / (n! (2n+1)!)
= 1/(2n+1) * (3n)! / (n! (2n)!)
= 1/(2n+1) * binom(3n,n).
```

Therefore:

```text
[x^n]Y = 1/(2n+1) * binom(3n,n).
```

Since `C_3=1+Y`, the constant coefficient is `FC_0^(3)=1`, and for all `n >= 1`:

```text
[x^n]C_3(x)=FC_n^(3)=1/(2n+1) * binom(3n,n).
```

This is the standard Fuss-Catalan coefficient extraction. Standard references for the Lagrange inversion / Fuss-Catalan calculation include Stanley, *Enumerative Combinatorics*, Vol. 2, and Flajolet-Sedgewick, *Analytic Combinatorics*, Section I.6. This theorem records only the inline P3.b calculation; bibliography hardening remains a separate deferred item.

V1 is used only as a sanity check, not as the primary closure mechanism:

```text
FC_0^(3)=1,
FC_1^(3)=1,
FC_2^(3)=3,
FC_3^(3)=12,
FC_4^(3)=55,
FC_5^(3)=273.
```

The V2 structural recursion supplies the generating-function equation. The V3 coefficient extraction supplies the general Fuss-Catalan identity.

## 3. S1 — Statistic well-definedness

The constructor-shape statistic is:

```text
k_t(T)=number of t-nodes in T.
```

It is well-defined on alpha-classes of eta-long beta-normal de Bruijn lambda terms of type `C_A^(3)`.

Alpha-equivalence respect: de Bruijn representation removes dependence on variable names. Renaming bound variables changes no constructor nodes. Therefore `k_t` is invariant under alpha-equivalence.

Finiteness: every normal form generated by the grammar is a finite expression built by finitely many applications of `t` to subterms, with base case `x`. Therefore `k_t(T)` is a finite nonnegative integer for each normal form.

Compatibility with the underlying canonical statistic: the manuscript records:

```text
sigma_C(s)=#lambda-nodes + #application-nodes + #variable-leaves.
```

The constructor-shape statistic used in `C_3(x)` is a projection of `sigma_C`, not raw `sigma_C` on the underlying lambda term on the nose. If:

```text
sigma_C = L + A + V
```

where `L` is lambda-node count, `A` is application-node count, and `V` is variable-leaf count, then on the restricted P3.a grammar the statistic `k_t` is the arity-three constructor-application component of `A`: it counts the applications headed by the ternary constructor `t` and ignores the fixed lambda-skeleton and variable-leaf contributions. Under the grammar `T ::= x | t(T,T,T)`, the only constructor applications in the constructor-shape layer are these `t`-applications. This matches the A1 paradigm, where the Catalan series enumerates binary constructor applications in:

```text
T ::= x | b(T,T)
```

rather than the raw lambda-node plus application-node plus variable-leaf total of the underlying eta-long beta-normal lambda term.

Thus S1 is verified within the normalized `HG-FND-002` discipline.

## 4. S2 — Formal-series enumeration

P3.b closure verifies:

```text
C_3(x)=sum_{n >= 0} a_n x^n
```

where `a_n` is the number of alpha-classes of eta-long beta-normal terms of type `C_A^(3)` with exactly `n` ternary constructor nodes.

The structural enumeration has two cases.

First, the variable-leaf term `x` has constructor count `0`. It contributes the constant term:

```text
1.
```

Second, every non-base term is uniquely of the form:

```text
t(T1,T2,T3)
```

where `T1`, `T2`, and `T3` are terms generated by the same grammar. This contributes one top `t`-node and three independent subterm enumerators, hence:

```text
x C_3(x)^3.
```

Summing the two disjoint cases gives:

```text
C_3(x)=1+x C_3(x)^3.
```

Applying Lagrange inversion to:

```text
Y=x(1+Y)^3,
C_3=1+Y,
```

gives:

```text
a_n=FC_n^(3)=1/(2n+1) * binom(3n,n).
```

The first coefficients are:

```text
C_3(x)=1+x+3x^2+12x^3+55x^4+273x^5+...
```

These are the arity-three Fuss-Catalan coefficients under constructor-shape enumeration. Therefore S2 is verified.

## 5. S3 — Analyticity at origin

Define:

```text
F(x,y)=y-1-x y^3.
```

At `(x,y)=(0,1)`:

```text
F(0,1)=0,
partial F / partial y = 1 - 3x y^2,
(partial F / partial y)(0,1)=1.
```

The analytic implicit function theorem therefore gives a unique analytic function `C_3(x)` in a neighborhood of `x=0` such that:

```text
F(x,C_3(x))=0,
C_3(0)=1.
```

Equivalently:

```text
C_3(x)=1+x C_3(x)^3,
C_3(0)=1.
```

The coefficients extracted above are finite nonnegative integers, and the algebraic equation defines a nonzero radius of convergence for the selected analytic branch. S3 is verified.

This theorem closes existence of a positive radius of convergence for the selected branch. It does not locate the radius at P3.c-grade. The traceability value `rho_3=4/27` remains downstream to P3.c / `HG-MTH-018`.

## 6. S4 — Continuation discipline

`C_3(x)` admits algebraic continuation as a branch of the cubic algebraic curve:

```text
y=1+x y^3.
```

Equivalently:

```text
x y^3 - y + 1 = 0.
```

The P3.b closure uses the branch selected near the origin by:

```text
C_3(0)=1.
```

This satisfies the continuation discipline required at the `HG-FND-002` level: the generating function is not merely a finite enumeration table; it is specified as a formal power series at the origin and as a local analytic branch of an algebraic equation.

The full analytic-continuation structure of this branch, including branch points, Puiseux expansion, monodromy, and finite phase reduction, is not part of P3.b. Those belong to P3.c and P3.d.

S4 is verified at the scope appropriate to P3.b.
