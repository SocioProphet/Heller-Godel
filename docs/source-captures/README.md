This directory captures the uploaded PDF/TeX/context corpus as markdown notes so the repository does not depend only on opaque PDFs.

These are curated markdown captures, not full verbatim OCR replacements. They preserve the load-bearing concepts, claim boundaries, examples, equations, and routing decisions. Full binary import remains tracked by HG-M1.
