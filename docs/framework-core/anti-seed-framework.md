# Framework-Core Anti-Seed Register

Status: active governance register.

## A-HG-FND-004 — Carry is not regulator data

Boundary: the finite section carry is not automatically the same object as a regulator-symbol construction. Separate typed comparison is required.

## A-HG-FND-005 — Tree enumeration is not typed proof grammar by default

Boundary: tree enumerations require typed fragment, witness convention, and statistic before framework citation.

## A-HG-FND-006 — Series is not proof-family generating function by default

Boundary: a series requires declared grammar, witness family, and statistic before citation through `HG-FND-002`.

## A-HG-FND-007 — Singularity is not decorated Puiseux datum by default

Boundary: decorated Puiseux datum requires chosen puncture, coordinate, branch convention, and local expansion.

## A-HG-FND-008 — Deck-character product is not carry triviality

Boundary: finite-character multiplication and integer-lift carry are different structures. A trivial product may still have nonzero carry.

## A-HG-FND-009 — Section carry is not comparison data by default

Boundary: `HG-FND-007` owns integer-valued section-defect carry only. Any comparison with Deligne, tame-symbol, regulator-symbol, or related analytic data requires a separate typed bridge.

## A-HG-VOC-006 — Character value requires source declaration

Boundary: character citation requires source, generator convention, and manuscript alignment.

## A-HG-MTH-003 — Citation grade must be declared

Boundary: fixture-grade, method-grade, and theorem-grade citations must remain distinct.

## Register maintenance

Entries are append-preserving and may be closed only by a later structural correction.
