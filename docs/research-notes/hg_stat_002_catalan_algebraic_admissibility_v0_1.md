# HG-STAT-002: Catalan Algebraic Test for Admissible Statistics

## A Successor Note to HG-STAT-001 v0.1

**Status:** v0.1 — theorem-level for the Catalan-grounded type under both syntax conventions; refines the strong conjecture of HG-STAT-001 by refuting radius-invariance under positive-weight reweighting.

---

## Abstract

HG-STAT-001 v0.1 established theorem-level robustness for two rational rank-uniform calibration families, chain and `((A -> A) -> A) -> A`, under admissible constructor-linear statistics: analytic type, dominant radius, and dominant local exponent are all preserved across positive-weight rescaling.

This note tests the corresponding question on the algebraic Catalan-grounded type:

```text
C_A = (A -> A -> A) -> A -> A.
```

The result is two-sided:

- **Positive theorem.** Across admissible weights, `T_{C_A}^{sigma_w}` remains algebraic with dominant local Puiseux exponent `alpha = 1/2`. Both invariants are preserved.
- **Negative result.** The dominant radius `|rho|` is not invariant under positive-weight reweighting; instead, it transforms covariantly as `|rho_w| = 4^(-1/a)` where `a` is a convention-dependent linear functional of the weights.

This refutes the rank-uniformity version of the strong conjecture in HG-STAT-001 §7.2: rank-uniformity is not sufficient for radius preservation. The chain and T-family preservation of `|rho| = 1` was a special case in which the singular schema `(1 - x^k)` has all roots on the unit circle for any `k`. Catalan's singular schema `(1 - 4*x^a)` has root at `4^(-1/a)`, which depends on `a`.

The refined picture: **radius covaries under monomial regrading induced by weight rescaling, rather than being invariant**. This is consistent with standard analytic-combinatorics behavior under change of size variable.

This note does not patch HG-STAT-001 v0.1 or CIPC v2.1.3. It is a successor result that opens a sharper conjectural frontier.

## 1. Background

HG-STAT-001 v0.1 established theorems for the chain and `((A -> A) -> A) -> A` calibration families:

- Both families are rank-uniform regular grammars: every inhabitant at level `n` has the same node-class profile.
- Under admissible constructor-linear statistics

```text
sigma_w = w_lambda*|lambda(s)| + w_app*|app(s)| + w_var*|var(s)|
```

with positive weights, the resulting generating functions are rational with `|rho| = 1` and `alpha = -2`.

HG-STAT-001 §7.1, the weak conjecture, extended this to analytic-type preservation across regular/algebraic/D-finite proof grammars. HG-STAT-001 §7.2, the strong conjecture, further suggested radius and exponent preservation under rank-uniformity.

HG-STAT-001 listed the next bounded test as: verify on the Catalan-grounded type `C_A`, the standard algebraic calibration. This note carries out that test.

## 2. The Catalan-grounded type `C_A`

Define:

```text
C_A = (A -> A -> A) -> A -> A.
```

Normal beta-normal eta-long inhabitants of `C_A` are:

```text
lambda f. lambda x. body
```

where `body` is generated by:

```text
body ::= x | f body_1 body_2.
```

The number of inhabitants at level `k`, where `k` is the number of internal `f`-applications, is the `k`-th Catalan number:

```text
C_k = 1/(k+1) * binomial(2k,k).
```

For example:

```text
C_0 = 1
C_1 = 1
C_2 = 2
C_3 = 5
C_4 = 14
C_5 = 42
```

## 3. Syntax conventions

Constructor counts depend on how applications are tokenized in the de Bruijn syntax tree. We treat both conventions explicitly.

### Convention A: binary application

Each expression `f body_1 body_2` is a single application node carrying two arguments. The head of this node is the variable `f`. Bare variable occurrences are nodes with empty argument list.

Under Convention A, each inhabitant at level `k` has profile:

```text
|lambda| = 2
|app|    = k
|var|    = 2k + 1
```

Reason:

- two outer lambdas, for `f` and `x`;
- `k` application nodes, one per internal `f`-call;
- `2k+1` variable occurrences: `k` instances of `f` as application heads plus `k+1` leaf occurrences of `x`.

### Convention B: curried application

Each expression `f body_1 body_2` is parsed as `((f body_1) body_2)`, i.e. two strictly binary application nodes: the inner applies `f` to `body_1`, and the outer applies the resulting term to `body_2`.

Under Convention B, each inhabitant at level `k` has profile:

```text
|lambda| = 2
|app|    = 2k
|var|    = 2k + 1
```

Reason:

- two outer lambdas;
- two application nodes per internal `f`-call;
- `2k+1` variable occurrences: `k` instances of `f` and `k+1` leaf occurrences of `x`.

Both conventions are rank-uniform: at fixed `k`, every Catalan inhabitant has the same node-class profile.

## 4. Theorems

Let `C(y)` denote the Catalan generating function:

```text
C(y) = (1 - sqrt(1 - 4y)) / (2y).
```

### Theorem 4.1: Catalan robustness, Convention A

Under Convention A and admissible weights `w = (w_lambda, w_app, w_var)` with all weights positive,

```text
T_{C_A}^{sigma_w}(x) = x^{b_A} * C(x^{a_A})
```

where:

```text
a_A = w_app + 2*w_var
b_A = 2*w_lambda + w_var.
```

Consequences:

- analytic type is algebraic over `Q(x)`, preserved across the admissibility class;
- dominant Puiseux exponent at the leading singularity is `alpha = 1/2`, preserved across the admissibility class;
- dominant singular radius is `|rho_w| = 4^(-1/a_A)`, which depends on weights and is not preserved.

**Proof.** From Convention A, an inhabitant at level `k` has statistic:

```text
sigma_w = 2*w_lambda + k*w_app + (2k+1)*w_var
        = (w_app + 2*w_var)*k + (2*w_lambda + w_var)
        = a_A*k + b_A.
```

There are `C_k` inhabitants at level `k`. Therefore:

```text
T_{C_A}^{sigma_w}(x)
  = sum_{k>=0} C_k * x^{a_A*k + b_A}
  = x^{b_A} * sum_{k>=0} C_k * (x^{a_A})^k
  = x^{b_A} * C(x^{a_A}).
```

Since `C(y)` is algebraic, `C(x^{a_A})` is algebraic. Its dominant singular equation is `x^{a_A} = 1/4`, hence `|rho_w| = 4^(-1/a_A)`. The local square-root exponent remains `1/2` under the monomial substitution. QED.

### Theorem 4.2: Catalan robustness, Convention B

Under Convention B and admissible weights,

```text
T_{C_A}^{sigma_w}(x) = x^{b_B} * C(x^{a_B})
```

where:

```text
a_B = 2*w_app + 2*w_var
b_B = 2*w_lambda + w_var.
```

Consequences are the same: algebraic type preserved, exponent `1/2` preserved, dominant radius `4^(-1/a_B)` weight-dependent and therefore not invariant.

**Proof.** From Convention B, an inhabitant at level `k` has statistic:

```text
sigma_w = 2*w_lambda + 2k*w_app + (2k+1)*w_var
        = (2*w_app + 2*w_var)*k + (2*w_lambda + w_var)
        = a_B*k + b_B.
```

There are `C_k` inhabitants at level `k`, so the same summation gives `x^{b_B} * C(x^{a_B})`. QED.

## 6. Refinement of HG-STAT-001 conjectures

The Catalan result refutes HG-STAT-001's strong conjecture (§7.2) as stated. Rank-uniformity is not sufficient for radius preservation: Catalan is rank-uniform, but its radius shifts under positive-weight regrading.

The chain and T-family preservation of `|rho| = 1` in HG-STAT-001 was a special case of singular schema `(1 - x^k)`, all of whose roots lie on the unit circle. Catalan's singular schema `(1 - 4*x^a)` exposes the general covariance.

### Conjecture 6.1: analytic type preservation

For regular/algebraic/D-finite proof grammars, admissible constructor-linear statistics preserve analytic class.

Catalan supports this conjecture in the algebraic setting.

### Conjecture 6.2: exponent preservation under rank-uniformity

For rank-uniform proof grammars, admissible constructor-linear statistics preserve the dominant local Puiseux exponent at the leading singularity.

This holds for chain, T-family, and Catalan.

### Conjecture 6.3: radius covariance under monomial regrading

For rank-uniform proof grammars whose level-aggregated generating function has dominant singularity `rho_y` in a level variable `y`, positive-weight regrading induces a monomial substitution `y = x^{a(w)}`, hence:

```text
|rho_w| = |rho_y|^(1/a(w)).
```

Thus radius is covariant, not invariant.

For chain and T-family, `rho_y = 1`, so the covariance formula gives `|rho_w| = 1`. For Catalan, `rho_y = 1/4`, so it gives `|rho_w| = 4^(-1/a(w))`.

## 7. What this note establishes / does not establish

### Established

For `C_A` under admissible constructor-linear statistics, in both syntax conventions:

- analytic type is algebraic and preserved;
- dominant local Puiseux exponent is `1/2` and preserved;
- dominant radius covaries as `4^(-1/a)` and is not invariant;
- rank-uniformity alone is insufficient for radius preservation.

### Conjectural

- analytic type preservation for general regular/algebraic/D-finite proof grammars;
- exponent preservation under rank-uniformity;
- radius covariance in the general rank-uniform setting.

### Not addressed

- non-rank-uniform grammars;
- multi-shell character data `chi_p^(rho,beta)`;
- D-finite grammars;
- recognition-dynamical consequences.
